### Paired events allow organizations to run two events that simultaneously use the same scanner. 

## 

## How to pair an event
### 1. Create both of the events that you want to pair
### 2. Click on one of the events in the events tab
### 3. Click the settings wheel in the top right corner
### 4. Pair the event to the event you would like to link it to
